# esformatter-equals

[esformatter](https://github.com/millermedeiros/esformatter) plugin for
triple equals and not double equals.


## Usage

install it:

```sh
npm install esformatter-equals
```

and add to your esformatter config file:

```json
{
  "plugins": [
    "esformatter-equals"
  ],
}
```
